# Security

## Reporting Vulnerabilities

Email **contact@terronex.dev** instead of opening a public issue. We will respond within 48 hours.

## Security Model

- Engram Trace Lite is a pure computation library. It does not perform I/O, network calls, or file operations.
- LLM summarization (optional) is provided by the caller. The library itself makes no external calls.

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
